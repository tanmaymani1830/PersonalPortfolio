# PORTFOLIO WEBSITE

A personal portfolio website designed to showcase projects, technical skills, and contact information. This project highlights front-end development abilities and serves as an online professional presence.

## 🌐 Live Demo
Visit the live portfolio here:  
👉 https://karthik-004.github.io/PORTFOLIO/

## Overview
This portfolio website presents an overview of my background, skills, and projects in a clean and structured layout. It is built to be responsive, user-friendly, and easy to navigate across devices.

## Key Features
- Responsive design for desktop, tablet, and mobile devices
- Dedicated sections for projects, skills, and contact details
- Clean and modern UI layout
- Easy navigation and readable content
- Lightweight and fast-loading pages

## Technology Stack
- HTML5
- CSS3
- JavaScript

## Getting Started

### Prerequisites
A modern web browser (Chrome, Edge, Firefox, etc.)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/KARTHIK-004/PORTFOLIO.git
````

2. Navigate to the project directory:

   ```bash
   cd PORTFOLIO
   ```
3. Open `index.html` in your browser.
